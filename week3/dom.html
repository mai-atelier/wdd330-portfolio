DOM : The Document Object Model

The Document Object Model represents an HTML page as a tree node.

The root node : HTML tags

The child node : every other part such as texts, attributes, classes

üí° DOM stores any whitespaces as text nodes.

  • document.body
  • document.images
  • document.links used for <a href><area href>
  • document.anchors used for<a name>

particular names

  • document.getElementById()
  • document.getElementByClassName()

CSS query selectors

  • document.querySelector()
  • document.querySelectorAll()

üßê className vs classList

className : to give a name directly to an element that will overwrite all other classes has given

classList : a list all the classes of an element

The classList properties

  • add.( ' class ' ) : add an another class to an element
  • remove.( ' class ' ) : remove the class from an element
  • toggle.( ' class ' ) : automatically add or remove a class
  • contains.( ' class ' ) : check if an element has a class name

Creating elements

  1. Create the element nodes : document.createElement
  2. Create the text nodes : document.textNodes( ' texts ' )
  3. Append the element nodes : 1.textContent
