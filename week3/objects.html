Objects

What is an object?

It is a set of properties with related values and functions.

Where are they used in the real world?

It is used to store information such as names, users, orders, and so on.

How are they used?

"It keeps any related information and functionally together in the same place."

What are some of the benefits of use?

"It is created quickly without the need for defining a class."

Nested Objects

Is it possible to contain other objects?

Yes. It is called "nested objects."

How to access to the nested object values?

object.childObject.property || object[ 'childObject' ][ 'property' ] || object.childObject['property']

How to Access

Access the object

this.property

Access the properties

property.name

What happens if a property does not exist?

It returns undefined.
