Lesson 8

Transforms and Transitions

Transform functions

It apply the values in the order provided

{ transform: value1 value2 value3 }

  • translate
  • scale
  • rotate
  • skew

Translating

  • It can move elements without flow impact vs position property
  • does not work on inline-elements
  • add { display: inline-block} for inline-elements

{ transform: translate( x, y ); }

translateX( );

translateY( );

Scaling

  • It does not have a flow impact
  • It grow or shrink elements towards its center

transform: scale( x, y );

scaleX( );

scaleY( );

Rotation

rotate( deg)

  • specify the angle value in degrees moving to clockwise
  • rotates element around the center point as an origin
  • the default origin point is the center of the element

Skew

rotate( x, y )

skewX( );

skewY( );

transform-origin

transform-origin: 0 0;

  • controlling the origin
  • background-position: 0 0; (default value)

Transitions

  • less processing rather than using Java Script
  • appear smoothly especially on mobiles
  • can animate by switching property
  1. The original state of the element declaration in the default style
  2. The final state of the transitioned element declaration
  3. Include the transition functions in 1.

Specify the property

  • define which property to transition
  • the default value is all for using when animate every property
  • property1, property2 ;

Define the length of animate time

transition-duration:

  • set how long the transition will take
  • specify the time in seconds (s) or milliseconds (ms)
  • the optimum time is 200ms

Control pace

transition-timing-function:

  • adjust the speed of the transition

key terms

linear
ease
ease-in-out
ease-in
ease-out
  • üßê cubic-bezier

linear is the same as cubic-bezier(0, 0, 1, 1)

Adjust the time animate starts for users

transition-delay:

  • set how long delay the transition begins
  • the default is 0

{ transition: duration delay perform }

  • the values can be in any order
  • duration must come before the delay

Multiple transitions

{ transition-* : property1, property2; }

Animations

Create keyframes

  • insert steps between the animation start and end

@keyframe name {

from { }

to { }

}

  • specify how far the keyframe location along the animation

@keyframe name {

0% { }

100% { }

}

Assign a name

animation-name: name;

attach a name of the keyframe as a property

Declaring the duration

animation-duration:

  • allocate the time for the animation to complete
  • the default is 0s

animation-timing-function

  • linear
  • ease
  • ease-in
  • ease-out
  • cubic-bezier
  • step-start
  • step-end
  • steps(number, direction)

steps() function:

determine direction options

  • start
  • end

animation-iteration-count:

  • define how many times the animation will play through
  • infinite never ends repeating
  • the default is 1 that will play only once

animation-direction:

  • Animations complete at 100% keyframe and back to the 0% keyframe to play again
  • Playing animations reverse, also reverse timing functions

options

  • normal (default)
  • reverse
  • alternate
  • alternate-reverse

animation-delay:

  • define how long it will take to start the browser begins playing the animation

animation-fill-mode:

  • none (default)
  • forwards (0%)
  • backwards (100%)
  • both (0% play 100%)

animation-play-state:

  • running (default)
  • paused
